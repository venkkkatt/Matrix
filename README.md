Matrix
=

A college-focused social networking platform built using MERN Stack, that enables students to connect with peers, share posts, create communities, discover upcoming events, and engage actively within their campus ecosystem.

## Features
  - College-only registration
  - Post updates and media
  - Follow peers
  - User authentication and profiles
  - Personalized Home Feed
  - Create and manage Open/Private Communities
  - Share posts exclusively in communities
  - Event creation and RSVP-based participation 
  - Secure backend architecture with protected routes
  - Fully responsive design for mobile and desktop devices
